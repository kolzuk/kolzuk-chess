# kolzuk-chess

## Описание проекта

Бекенд для шахматного движка.

### Список работающих сервисов
- chess-engine

## chess-engine
- Запуск через `sbt "chessEngine/run"`

## Основные функции и возможности
Наш проект предоставляет бэкенд для шахматного движка, обеспечивая основу для разработки шахматных приложений.

## Инструкции по установке
Для запуска проекта вам потребуется установленный [SBT](https://www.scala-sbt.org/). После клонирования репозитория выполните:
```
sbt "chessEngine/run"
```

## Инструкции по использованию
Для запуска шахматного движка выполните команду `sbt "chessEngine/run"`. Это запустит движок в режиме разработки, и вы сможете начать работу с ним.

## Архитектура и основные компоненты
Проект состоит из нескольких компонентов, включая шахматный движок, который обрабатывает логику игры. Каждый компонент может быть расширен или модифицирован для добавления новых функций.
