# kolzuk-chess

## Описание проекта

### Общая структура
Проект представляет собой бекенд для шахматного движка. Он включает в себя несколько компонентов, обеспечивающих работу шахматного приложения.

### Основные файлы и папки
- `chess-engine` - папка, содержащая файлы шахматного движка.
- `README.md` - файл с описанием структуры проекта.

### Назначение ключевых компонентов
- **chess-engine** - основной компонент, который реализует логику шахматного движка. Для запуска используйте команду `sbt "chessEngine/run"`.

### Как начать работу с проектом
1. **Клонируйте репозиторий**: `git clone <ссылка_на_репозиторий>`.
2. **Перейдите в папку проекта**: `cd kolzuk-chess`.
3. **Запустите шахматный движок**: `sbt "chessEngine/run"`.
