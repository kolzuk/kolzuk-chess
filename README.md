# kolzuk-chess

## Введение

Бекенд для шахматного движка.

### Основные функции и возможности
- Поддержка запуска шахматного движка через удобный интерфейс.

### Установка

Для запуска проекта вам потребуется установить [SBT](https://www.scala-sbt.org/).

1. Клонируйте репозиторий:
   ```bash
   git clone https://ваш_репозиторий.git
   ```

2. Перейдите в директорию проекта:
   ```bash
   cd kolzuk-chess
   ```

### Использование

Для запуска шахматного движка выполните команду:
```bash
sbt "chessEngine/run"
```
