# kolzuk-chess

## Назначение проекта

Проект представляет собой бэкенд для шахматного движка, обеспечивающий основу для разработки и тестирования шахматных алгоритмов.

## Основные компоненты и их назначение

- **chess-engine**: Модуль, реализующий шахматный движок, который может быть использован для анализа шахматных позиций, поиска ходов и игры против компьютера.

## Основные команды для работы с проектом

- Запуск шахматного движка: `sbt "chessEngine/run"`
