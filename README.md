# kolzuk-chess

## Описание проекта

### Общее описание
Бекенд для шахматного движка. Проект включает в себя несколько сервисов, обеспечивающих работу шахматного движка.

### Структура проекта
Проект состоит из следующих основных компонентов:
- `chess-engine` - шахматный движок, который реализует логику игры в шахматы.

### Список работающих сервисов
- chess-engine

### Инструкции по установке и запуску проекта

#### Установка
Для работы с проектом вам потребуется установить [Scala и SBT](https://www.scala-sbt.org/).

#### Запуск
Для запуска шахматного движка выполните следующую команду:
```bash
sbt "chessEngine/run"
```

### Основные разделы и подразделы проекта
- `conf` - конфигурационные файлы.
- `src` - исходный код проекта.
  - `main` - основной код, который будет выполняться при запуске проекта.
  - `test` - тестовые файлы для проверки работоспособности кода.
- `project` - файлы конфигурации для сборки проекта с помощью SBT.

### Зависимости
Для работы проекта могут потребоваться следующие зависимости:
- [Scala](https://www.scala-lang.org/)
- [SBT](https://www.scala-sbt.org/)
- Дополнительные библиотеки для работы с шахматами (например, для представления шахматных позиций).
