# kolzuk-chess

## Общая информация

Бекенд для шахматного движка.

### Ключевые компоненты
- chess-engine

## Getting Started

Для запуска проекта выполните следующие шаги:
1. Установите [SBT](https://www.scala-sbt.org/download.html).
2. Клонируйте репозиторий: `git clone https://github.com/your-username/kolzuk-chess.git`.
3. Перейдите в директорию проекта: `cd kolzuk-chess`.
4. Запустите проект: `sbt "chessEngine/run"`.

## Features

- Шахматный движок, поддерживающий стандартные правила игры.

## How to Contribute

Если вы хотите внести свой вклад в проект, следуйте этим шагам:
1. Создайте форк репозитория.
2. Клонируйте ваш форк: `git clone https://github.com/your-username/kolzuk-chess.git`.
3. Создайте ветку для вашей задачи: `git checkout -b your-branch`.
4. Внесите изменения и закоммитьте их.
5. Создайте пулл-реквест в основной репозиторий.
