# kolzuk-chess

## Введение
Бекенд для шахматного движка.

## Установка
Для запуска проекта вам потребуется установить [SBT](https://www.scala-sbt.org/). 
Затем клонируйте репозиторий и перейдите в директорию проекта в терминале.

### Запуск через SBT
1. Клонируйте репозиторий: `git clone <ссылка_на_репозиторий>`
2. Перейдите в директорию проекта: `cd kolzuk-chess`
3. Запустите SBT: `sbt`

## Использование
Список работающих сервисов:
- chess-engine

### chess-engine
- Запуск через `sbt "chessEngine/run"`
