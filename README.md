# kolzuk-chess

## Overview

**Project Name:** kolzuk-chess  
**Description:** Backend for a chess engine.

## Project Purpose

The purpose of this project is to provide a robust backend for a chess engine, enabling it to be used in various applications and environments.

## Installation

To install the project, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/kolzuk-chess.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd kolzuk-chess
   ```

3. **Ensure you have `sbt` installed.** You can install `sbt` by following the instructions on the [official website](https://www.scala-sbt.org/).

## Usage

### chess-engine

- **Launch the chess engine:**
  ```bash
  sbt "chessEngine/run"
  ```

## Configuration

The configuration settings for the chess engine can be found in the `chessEngine/src/main/scala/config.scala` file.
