# kolzuk-chess

## Описание проекта и его цели
Kolzuk-chess — это бэкенд для шахматного движка, предназначенный для разработки и тестирования шахматных алгоритмов.

## Список основных функций и возможностей
- Поддержка запуска шахматного движка.

## Инструкции по установке и настройке
Для запуска шахматного движка выполните следующие шаги:
1. Убедитесь, что у вас установлен SBT.
2. Перейдите в директорию проекта.
3. Запустите движок командой `sbt "chessEngine/run"`.

## Краткое руководство пользователя
Чтобы запустить шахматный движок, выполните команду `sbt "chessEngine/run"`.
