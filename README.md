# kolzuk-chess

## Общая информация

Бекенд для шахматного движка. Проект предоставляет основу для разработки шахматных приложений и игр.

## Основные функции и возможности

- Поддержка работы с шахматным движком;
- Управление игровым процессом через удобный интерфейс.

### Список работающих сервисов

- chess-engine

## chess-engine

- Запуск через `sbt "chessEngine/run"`

## Инструкции по установке

Для запуска проекта необходимо выполнить следующие шаги:
1. Установить SBT (Scala Build Tool);
2. Клонировать репозиторий;
3. Перейти в директорию проекта;
4. Запустить `sbt "chessEngine/run"`.

## Инструкции по использованию

Для запуска шахматного движка выполните команду `sbt "chessEngine/run"`.
