## Getting Started

Для запуска проекта вам потребуется установить [Scala и SBT](https://www.scala-lang.org/ и https://www.lightbend.com/sbt).

1. Клонируйте репозиторий:
   ```bash
   git clone https://ваш_репозиторий.git
   ```

2. Перейдите в директорию проекта:
   ```bash
   cd kolzuk-chess
   ```

3. Для запуска chess-engine выполните команду:
   ```bash
   sbt "chessEngine/run"
   ```
