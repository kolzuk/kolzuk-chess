# kolzuk-chess

## Описание проекта и его целей

Проект kolzuk-chess представляет собой бэкенд для шахматного движка. Он разработан для обеспечения функциональности, необходимой для работы с шахматными партиями и анализа шахматных позиций.

### Цели проекта
- Разработка и поддержка эффективного шахматного движка.
- Предоставление API для взаимодействия с шахматным движком.

## Инструкция по установке

Для установки проекта вам потребуется:
1. Установленный SBT (Simple Build Tool) для работы с Scala проектами.
2. Доступ к репозиторию проекта.

Шаги по установке:
1. Клонируйте репозиторий проекта: `git clone <ссылка_на_репозиторий>`.
2. Перейдите в каталог проекта: `cd kolzuk-chess`.
3. Установите зависимости проекта: `sbt update`.

## Инструкция по использованию

Для запуска шахматного движка выполните следующие шаги:
1. Запустите проект в режиме разработки: `sbt "chessEngine/run"`.

## Список основных функций и возможностей

- Запуск шахматного движка через команду `sbt "chessEngine/run"`.
- Взаимодействие с шахматным движком через API.
