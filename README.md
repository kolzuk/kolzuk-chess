# kolzuk-chess

## Описание проекта

Бекенд для шахматного движка.

### Основные функции и возможности
- Реализация бэкенда для шахматного движка, обеспечивающего логику игры.

### Список работающих сервисов
- chess-engine

## Установка и запуск проекта

Для запуска проекта выполните следующие шаги:
1. Установите `sbt` (если еще не установлен).
2. Перейдите в директорию проекта.
3. Запустите `sbt` и выполните команду `sbt "chessEngine/run"` для запуска chess-engine.

## Инструкции по использованию основных компонентов

### chess-engine
- Запуск через `sbt "chessEngine/run"`
