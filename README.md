# kolzuk-chess

## Описание проекта

Бекенд для шахматного движка.

### Список работающих сервисов

- chess-engine

## chess-engine

- Запуск через `sbt "chessEngine/run"`

## Установка и запуск проекта

Для запуска проекта выполните следующие шаги:
1. Установите [Scala и SBT](https://www.scala-sbt.org/download.html) (если ещё не установлено).
2. Клонируйте репозиторий: `git clone <ваш_репозиторий>`.
3. Перейдите в директорию проекта: `cd kolzuk-chess`.
4. Запустите проект: `sbt "chessEngine/run"`.

## Основные функции и возможности

- Реализация шахматных алгоритмов.
- Поддержка стандартных форматов хранения шахматных партий.
